# Roles & Permissions — SpaceDMV

---

## Civilian
- Register vehicles  
- Book appointments  
- View status  

## Examiner
- View assigned appointments  
- Approve/deny tests  

## DMV Officer
- Manage registrations  
- Oversee appointments  

## Insurance Agent
- Validate EVIN/SVIN data  

## Admin
- Full system access  
- User management  
- Logs & analytics
